The Bottom Line

Without IO-threads, Redis 8.6 wins by 6-9% across all pipelined tests.
With IO-threads enabled, Valkey 9.1 wins — peaking at 2.54M req/s, 15.5% faster than Redis 8.6.

For production workloads using IO-threads (recommended), Valkey 9.1 delivers the best performance — especially for write-heavy and high-throughput pipelined workloads — while also offering an open-source BSD-3 license.
