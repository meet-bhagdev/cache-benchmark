The bottom line.

Without IO‑threads, Redis 8.6 wins every pipelined test by 6–9%.

With IO‑threads enabled, Valkey 9.1 takes the lead — peaking at 2.54M req/s, 15.5% faster than Redis 8.6.

For production workloads using IO‑threads (which you should be), Valkey 9.1 delivers the best performance — especially for write‑heavy and high‑throughput pipelined workloads — while offering a fully open‑source BSD‑3 license.
